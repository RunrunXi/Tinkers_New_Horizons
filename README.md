---

## ✨ 特性概览

### 新增材料
- **充能合金**
  - 带点钴的属性，算是下位替代
  - 提高攻击速度，挖掘速度，拉弓速度
  - 适用于工具和远程
  - 来源：将金浇筑在红石上
- **龙蒿**
  - 石化树皮正在丛林...（不是
  - 和生命相关！在工具和武器上可以提升玩家生命上限，在护甲上甚至可以复活！
  - 适用于工具和护甲
  - 来源：丛林群系的基岩层，注意：这个矿需要下界合金工具和烈焰血
### 新增机制
- 生命赐福：可以消耗生命上限复活！
